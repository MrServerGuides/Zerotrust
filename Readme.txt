ZEROTRUST Lite v2

ZEROTRUST Lite v2 is a compact and efficient version of the ZeroTrust encryption system, designed for secure communication and data handling with minimal setup. It operates under a principle of least privilege and assumes that every connection, device, and user is untrusted until verified.

## Features

- **End-to-End Encryption**: Ensures data is encrypted at the sender's side and decrypted only at the recipientâ€™s end.
- **Lightweight & Fast**: Optimized for performance, ideal for mobile and embedded systems.
- **No External Dependencies**: Requires only native utilities; no need for additional software or libraries.
- **User-Friendly Interface**: Minimalist design with clear, simple commands.
- **Secure Key Exchange**: Includes built-in key verification and secure transfer protocols.
- **Compatibility**: Works seamlessly on Android (Pydroid 3 / Termux) and Linux systems.

## Usage

- Follow on-screen instructions to:
  - Generate encryption keys
  - Send encrypted messages
  - Receive and decrypt data
- Automatically logs session activity (optional feature)

## Safety & Privacy

ZEROTRUST Lite v2 does not collect or share any user data. All operations are performed locally unless explicitly configured otherwise.
